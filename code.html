devon4net

.Net stack on devonfw platform

devon4net (part of devonfw platform) is a modularized solution that provides advanced features for industrial applications.

Each feature is divided in nuget packages in order to provide a scalable .net core solution.

You can get all the resources at Github and Nuget site:

Github | Nuget

.Net Core + Entity Framework Core

devon4net is designed under the new and powerfull .net core platform.

Entity Framework for .net core is fully supported on devon4net. Sync and Async methods also are provided. Lazy data loading by demand is also included.

.NET platform | Entity Framework

Cross-Origin Resource Sharing - CORS

Cross-Origin Resource Sharing (CORS) is a mechanism that uses additional HTTP headers to tell a browser to let a web application running at one origin (domain) have permission to access selected resources from a server at a different origin.

devon4net provides by default multiple domain CORS features.

CORS

Docker

A Docker container image is a lightweight, standalone, executable package of software that includes everything needed to run an application: code, runtime, system tools, system libraries and settings.

devon4net offers fully compatibility with Linux containers. That means you can distribute multiplatform application tha runs on Windows, Linux and iOS systems.

Docker

Swagger

Swagger offers the most powerful and easiest to use tools to take full advantage of the OpenAPI Specification.

devon4net provides swagger support by default. Swagger allows share your API contract to multiple teams.

Swagger

Graylog

Graylog provides log data available and accessible in one central location.

This feature is available, and it is very useful on microservices environment or distributed systems.

Graylog

JSON Web Tokens - JWT

JSON Web Tokens are an open, industry standard RFC 7519 method for representing claims securely between two parties.

devon4net provides JWT support by default enabling securized communications between client and server. An example of this could be Angular and mobile applications with devon4net backend.
